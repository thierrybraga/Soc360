Sobre o Open CVE

Solução aberta para geração de relatórios inteligentes e análise automatizada de vulnerabilidades

🧠 Como funciona?

O Open CVE coleta dados públicos de vulnerabilidades (CVEs) diretamente da base oficial do NIST por meio de APIs atualizadas.

Essas informações são armazenadas localmente em um banco de dados otimizado, permitindo consultas por fornecedor, ano, severidade e outros critérios técnicos.

⏱ Atualizações Automáticas

Com um agendador de tarefas interno, o sistema realiza atualizações periódicas com os CVEs mais recentes, garantindo que você esteja sempre informado.

📄 Relatórios com Inteligência Artificial

Utilizando a API da OpenAI, o Open cve gera relatórios técnicos automaticamente a partir da descrição de vulnerabilidades. Os relatórios são gerados em Markdown e convertidos em PDFs profissionais prontos para uso em auditorias, consultorias e apresentações.
